# -Credit-Card-Transactions
This project analyzes credit card transaction data to identify spending patterns, transaction trends, and insights using SQL.
## Tools & Technologies
- SQL (MySQL)
- CSV Dataset
- GitHub

## Dataset
- credit_card_transcations.csv  
Contains transaction details such as amount, transaction type, date, and customer information.

## Analysis Performed
- Total transaction amount calculation
- High-value transaction identification
- Customer spending behavior analysis
- Date-wise transaction trends

## Files Included
- Portfolio Project 1 - Credit Card Transactions.sql
- credit_card_transcations.csv

## How to Use
1. Import the CSV file into MySQL.
2. Run the SQL script to perform analysis.
3. Review the results for insights.
